# AI Usage Notes

This file documents the usage of AI tools (e.g., GitHub Copilot, ChatGPT, Claude) during the development of the Personal Budget Manager application.

## Transparency & Reflection

* **Testing:** AI assistance was heavily utilized to generate robust unit tests for the business logic. It helped in covering edge cases and structuring the test suites effectively.
* **Frontend Development (HTML & Views):** AI played a significant role in creating and structuring the frontend views. It assisted with generating HTML templates, organizing the layout, and ensuring the user interface correctly visualizes the data provided by the backend API.
* **Documentation:** The initial structure and formatting of the `README.md` file were drafted with the help of AI, ensuring that all project requirements, run instructions, and endpoint descriptions were clearly articulated and easy to read.
